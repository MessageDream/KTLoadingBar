# KTLoadingBar
a loading bar used for ios

to show:
``` objc
    [KTLoadingBar showLoadingWith:self.contentView atPosition:KTLoadingBarPosition_Top];
```
to dismiss:
``` objc
    [KTLoadingBar showLoadingWith:self.contentView atPosition:KTLoadingBarPosition_Top];
```  
or:
``` objc
     [KTLoadingBar dismiss];
```
